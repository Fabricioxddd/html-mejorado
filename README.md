# Pagina web - Pruebas 3

Este proyecto es una aplicación web en desarrollo para evitar la desercion de los estudiantes universitarios

## Como ejecutarlo 
* **Usa Vs CODE y descargas la extensión live server.
* **Una vez instalada le das click derecho y te aparecera una opcion que dice "open whit live server".
* **Una vez presiones eso te dirigira a tu navegador.
* **Si lo ejecutas por github, abre la terminal y pones python3 -m http.server 8080
* **ingresa en modo invitado por el momento hasta que la pagina se publique oficialmente.

## 🚀 Funcionalidades
* **Login con Google:** Utilize una firebase para que se puedan registrar con goggle(por el momento no activa para el publico)
* **Dashboard interactivo:** Interfaz diseñada en HTML para mostrar información tras el inicio de sesión.

## 🛠️ Tecnologías utilizadas
* **Frontend:** HTML5, CSS3, JavaScript (ES6+).
* **Backend/Auth:** Integración con APIs de autenticación.

## 📂 Estructura del proyecto
* `index.html`: Pantalla principal de acceso.
* `dashboard.html`: Panel de control tras el login.
